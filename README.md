# jekinsAccount
该工具用于检测当前jekins站点是否可以进行账户注册，并通过注册账户访问/script


## 参数讲解

```
  -url URL        测试站点的url
  -file FILE      导入ip文件进行扫描（请注意，文本的条目应该是"IP:port"的形式）
  -user USER      自定义登录的用户名（默认yangsir）
  -passwd PASSWD  自定义登录的密码（默认123）
```
